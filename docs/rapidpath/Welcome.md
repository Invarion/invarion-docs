# Welcome to RapidPath

RapidPath tests vehicle movements for road and site projects, helping you check whether a planned layout can safely accommodate expected vehicle manoeuvres.

If you are looking for how to download and run RapidPath, see [Install and use the Invarion Launcher](/accounts-setup/desktop-applications/install-and-use-the-invarion-launcher).

This section covers RapidPath-specific **swept path analysis** functionality. For shared drawing, plan management, mapping, import/export, **workspace**, and general plan-editing tools, use the RapidPlan documentation.

In practice, many current RapidPath workflows depend on shared RapidPlan features such as basemaps, Spatial Data Import, **print regions**, and export. See [Shared RapidPlan Docs](/rapidpath/shared-rapidplan-docs) for the best entry points.

Sign-related functionality is available in RapidPath only when a signs add-on is assigned to your license. If you have a signs add-on, use the RapidPlan signs documentation for those tools.
