# Welcome to RapidPath

**RapidPath** tests vehicle movements for road and site projects, helping you check whether a planned layout can safely accommodate expected vehicle manoeuvres.

This section covers RapidPath-specific swept path analysis functionality. For shared drawing, plan management, mapping, import/export, workspace, and general plan-editing tools, use the RapidPlan documentation.

Sign-related functionality is available in RapidPath only when a signs addon is assigned to your license. If you have a signs addon, use the RapidPlan signs documentation for those tools.
