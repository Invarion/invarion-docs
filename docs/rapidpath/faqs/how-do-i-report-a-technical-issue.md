# How do I report a technical issue?

If RapidPath shows an error, crashes, or behaves unexpectedly, contact [Invarion Technical Support](https://invarion.com/support/).

For non-trivial technical issues, Technical Support may ask you to run Launcher Diagnostics. Launcher Diagnostics can collect Launcher logs, RapidPath logs, crash logs, environment details, connectivity checks, and related information into a single report.

See [Run Launcher Diagnostics](/accounts-setup/desktop-applications/run-launcher-diagnostics).
