# Creating your own signs

RapidPlan can create new signs from inside the application. Basic users may only need to create signs for a plan, while advanced users may need different versions of each sign with sign codes, multiple color schemes, and other variations.

Creating and saving a sign takes some practice. This article is recommended for users who are comfortable with objects, grouping, ungrouping, loading, and saving files.

***RapidPlan signs are made in RapidPlan***

That's right - every sign inside RapidPlan has been created from within the program, so you can use the same tools we did to create your own signs. It also means that you can save valuable time by simply editing the existing signs in the package to make new ones.
