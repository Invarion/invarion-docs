# Welcome to RapidPlan

**RapidPlan's** unique array of intuitive tools make short work of even the most complex plans – whether you are rerouting traffic for a marathon, or managing a temporary traffic work zone.

RapidPlan and RapidPath share many drawing, plan-management, mapping, import/export, and workspace tools. RapidPath users can use this RapidPlan documentation for shared functionality. RapidPath-specific swept path analysis features are documented in the RapidPath section.
