# Export formats

RapidPlan supports several output formats. Choose the format based on what the recipient needs to do with the file.

## PDF

Use PDF for most review, approval, email, and print-ready workflows.

PDF is usually the right format when you need:

- a stable page layout
- print frames and titleblocks
- one file per plan or stage
- output that is easy to share outside RapidPlan

## Image

Use image export when another system needs raster output or when you need to attach plan output as an image.

If the output needs spatial reference information, see [Georeferenced image export](./georeferenced-image-export.md).

## CAD

Use CAD export when the recipient needs editable drawing data or spatially aligned CAD output.

CAD export supports `DWG` and `DXF`. See [CAD export](./cad-export.md).

## Manifest formats

Manifest output can be exported to document and machine-readable formats where supported, including formats such as PDF, text, CSV, XML, and JSON.
