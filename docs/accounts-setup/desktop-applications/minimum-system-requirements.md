# Minimum system requirements

These are the minimum requirements for running Invarion **desktop applications** such as **RapidPlan** and **RapidPath**.

## Hardware requirements

- 64-bit processor
- 8 GB RAM
- 4 GB free disk space

These values are a bare minimum. For a comfortable experience when editing complex plans, use a more modern hardware configuration.

## Software requirements

- Windows 11
- **Invarion Launcher**

Install the latest version of the **Invarion Launcher** before opening **RapidPlan** or **RapidPath**. The Launcher installs and updates the Invarion **desktop applications** assigned to your account.

See [Install and use the Invarion Launcher](./install-and-use-the-invarion-launcher.md).

## Running on a Mac

**RapidPlan** and **RapidPath** are Windows **desktop applications**. They may run on a Mac through a Windows environment such as Parallels, but Invarion **Technical Support** does not provide support for installing or configuring that Windows environment, or for issues caused by it.
