# Welcome to RapidPlan Online

**RapidPlan Online** is a web-based application that enables you to draw detailed **traffic management plans** directly onto **integrated mapping** through a web browser.

**RapidPlan Online** and **RapidPath** Online share many cloud, **workspace**, drawing, plan-management, mapping, and export tools. **RapidPath** Online users can use this **RapidPlan Online** documentation for shared functionality. **RapidPath** Online-specific **swept path analysis** features are documented in the **RapidPath** Online section.
