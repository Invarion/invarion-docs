Toolbar

The Toolbar, located in the top right corner of the workspace, contains a number of button available for quick access. Each button is described in Table 4.2

LabelButtonDescription
PrintOpen the printing dialogue to print or export a plan.
Location featuresLocation-based features, including: Go to location and Toggle location marker.
Map viewToggle between different map views, including: Satellite, Road, Hybrid and Terrain.
ToolsShow/hide the Tools palette.
OperationsShow/hide the Operations palette.
